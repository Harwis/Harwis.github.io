# Why python? 
If you are a students, engineer, scientist or even an artist, and you want to learn a programming language, no doubt, you should pick up Python. There are many reasons as below:  
- Simple, easy to learn, readable  
- It is available on Windows, macOS, and Unix operating systems,  
- Large standard library and active community  
- Versatile, can be used for a wide range of applications, including web development, data analysis, artificial intelligence, scientific computing, automation, and more. 
- Free of use 
 
The statistics from HitHub and Google shows that Python has become the post popular programming language in 2020, and its popularity is keep on increasing.
